# Go-Tutorial

#### Inicializar um módulo

```bash
go mod init <module-name>
```

#### Instalar dependências

```bash
go mod tidy
```

#### Executar um arquivo `.go`

```bash
go run main.go
```

#### Compilar um arquivo `.go`

```bash
go build main.go
```

#### Executar um arquivo `.go` compilado

```bash
./main
```
